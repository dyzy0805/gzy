# gzy
A git logogram by node.js

## gzn ${option} ${message}
全局提交 git add . + git commit -m + git push

## gzn-d ${branchName}
全局删除分支 git branch -d + git push origin --delete

## gzn-c ${branchName}
全局创建分支 git checkout -b + git push --set-upstream origin
